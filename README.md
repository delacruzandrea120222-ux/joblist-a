# JobList Laravel Project

This is a simple Job Listing web application built using Laravel.

## Features
- View job listings
- Fake IT job data using factories
- Clean UI display
- MySQL database integration

## Technologies Used
- Laravel
- PHP
- MySQL
- Blade Template

## How to Run
1. Clone repository
2. Run composer install
3. Copy .env file and set database
4. Run php artisan migrate
5. Run php artisan db:seed (or factory)
6. Run php artisan serve
